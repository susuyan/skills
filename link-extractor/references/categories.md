# 分类板块详解

## 五大板块


### 🤓 Fav Finds
- **内容**: 当日最值得收藏的资源
- **样式**: 概念式两行 / 图标单行

### 📘 Read This
- **内容**: 文章、长文、官方博客
- **样式**: 概念式 / 要点式 / 官方博客

### 🛠️ Tools
- **内容**: 产品、服务、CLI、代码仓库
- **样式**: 图标单行 / 概念式 / GitHub Repo

### 🔧 Try This
- **内容**: 教程、实验、可实践方法
- **样式**: 概念式两行 / 要点式

### 🎧 Listen To
- **内容**: 播客、音频
- **样式**: 概念式两行

## 分类映射规则

| 内容类型 | 目标板块 | 理由 |
|---------|---------|------|
| 官方博客 | 📘 Read This | 深度阅读 |
| 技术文章 | 📘 Read This | 需要理解 |
| 开源项目 | 🛠️ Tools | 工具使用 |
| 产品/服务 | 🛠️ Tools | 可用资源 |
| 方法论/思考 | 🤓 Fav Finds | 值得收藏 |
| 教程 | 🔧 Try This | 需要实践 |
| 实验/示例 | 🔧 Try This | 可操作 |
| 播客/音频 | 🎧 Listen To | 听觉学习 |
| 生态资源 | 🤓 Fav Finds | 值得深挖 |

## 决策指南

### 模糊时的判断标准
- **重在"学习"** → 📘 Read This
- **重在"实践"** → 🔧 Try This
- **重在"使用"** → 🛠️ Tools
- **重在"收藏"** → 🤓 Fav Finds
- **重在"记忆"** → 🪶 Remember This

### 多价值点的内容
- 选择主要价值归类
- 其他价值在描述中体现

**示例**:
```markdown
# 主要是工具,但也包含教程
## 🛠️ Tools

- [Cursor](https://cursor.com) — AI-powered code editor
- 内置 agent 模式,支持 multi-file 重构(详见官方教程)
```

## 去重策略
- 同一 URL 当日仅记录一次
- 重复出现时合并来源信息与描述
- 保留最完整的上下文

## 新增板块指引
1. 在模板中添加标题与图标
2. 定义适用的内容类型和样式
3. 更新分类映射规则
4. 如需专用样式,在 references 目录新增文档
